**あなたのAIは、肉切り包丁で複雑なシステムを断ち割っている。料理人の刃を教えよう。**

絡み合ったコードベースや、曖昧な「Xを作って」という課題に直面したとき、たいていのAIは突進し、いきなり手を出す——構造を理解する前にファイルを編集し、最も密結合した核心から斬りかかり、どこでも同じ速さで振り回す。骨を断ち割ってしまう。刃は鈍る。ものは壊れる。

**庖丁解牛**(PaoDing JieNiu)は、荘子の伝説的な料理人が示した2300年前の答えを encode したものだ。その料理人の刃は十九年もの間、剃刀のように鋭いままだった——理に逆らって切ることがなく、もとからそこにある隙間を見つけ、刃を滑らせていったからである。

## 課題

```
あなた: 「この4万行のレガシーモジュールをリファクタリングして。」

庖丁なきAI:
  - 3つのファイルを開き、いきなり最も中心的なクラスを編集し始める
  - グローバル状態に触れ、12箇所を壊し、ロールバックできない
  - 暗号処理パスでもCSSファイルでも同じ無謀な速さ
  - 「いっそ全部書き直そう」(新刀又卷 — 新しい刃も、また鈍る)

庖丁あるAI:
  - まず構造を READ する:依存グラフ、データフロー、ドメイン境界 (依乎天理)
  - 隙間を見つける:純粋なモジュール、きれいなインターフェース、疎結合な層 (批大郤·导大窾)
  - 難しい結び目では速度を落とす——テストや小さな実験を加える (怵然为戒)
  - 小さく可逆なコミット。ツールチェーンと信頼は鋭いまま (刀刃若新)
```

## AIに何を教えるか

### 🔪 四つの底層原則 (四底层原则)

| 原則 | 中文 | AIの振る舞い |
|-----------|---------|--------------|
| 理に従う | 依乎天理 | 何かに触れる前に構造を読む |
| 隙間から入る | 批大郤·导大窾 | インターフェース、純粋なモジュール、疎結合な層から始める |
| 結び目では緩める | 怵然为戒·动刀甚微 | 核心・並行処理・セキュリティのパスでは減速する |
| 刃を鋭く保つ | 刀刃若新发于硎 | 小さく、可逆で、テストに裏打ちされた一歩——負債を溜めない |

### 🐂 牛の解剖学 (拆解概念体系)

このスキルは、AIに **どこを切るか** の語彙を与える:

| 用語 | 中文 | エンジニアリング上の意味 |
|------|---------|---------------------|
| 自然の理 | 天理 | システムに内在する構造の筋 |
| 大きな隙間 | 大郤 (xì) | 広く明白な境界——最初の切り口に最適 |
| 空洞 | 大窾 (kuǎn) | 関節と関節の間の隙間——刃が自在に流れる場所 |
| 硬い骨 | 大軱 (gū) | 核心の抽象/グローバル状態——力ずくで断ち割らない |
| 絡んだ結び目 | 族 (zú) | 密に結合した塊——ここでは速度を落とす |
| 遊びの余地 | 游刃有余 | 刃が常に保つ可逆性とゆとり |
| 神を以て遇う | 以神遇 | 構造を真に理解したとき、解牛は力みなく為される |

### 📏 解牛四歩法 (拆解四步法)

```
1. 观 OBSERVE  — 牛を地図化する:ディレクトリツリー、依存グラフ、データフロー、ドメイン
2. 寻 SEEK     — 隙間を見つける:最初の可逆でテスト可能な切り口
3. 慎 CARE     — 結び目では速度を落とす:テスト、小さな実験、退避口
4. 养 SUSTAIN  — 小さなコミット、常にグリーン、負債なし:刃を新たに保つ
```

### 以道驭术 — 力ではなく、道を以て

> 「良庖岁更刀，割也；族庖月更刀，折也。」
> 良き料理人は刃を年に一度替える——*切る*からだ。
> 並みの料理人は月ごとに替える——*断ち割る*からだ。
> — 荘子『養生主篇』

違いは、決して刃そのものにあるのではない。それを用いる **道(やりかた)** にある。庖丁解牛は **NoPUA** と同じ根を分かち持つ:*恐れではなく道で仕事を駆動し、パニックの力任せを構造の理解に置き換える。* 理に沿って切ることこそが、刃——あなたの道具、注意力、コンテキスト予算、そして信頼——を永遠に新たに保つ。

## 東西の出会い

庖丁解牛は、あなたのエンジニアリングの素養を置き換えるものではない。それを **研ぎ澄ます** のだ。

| 西洋の実践 | + 庖丁解牛 | = 完成 |
|------------------|-----------|------------|
| 「まず動かし、それからリファクタリング」 | まず 依乎天理 | 切る前に理解する——書き直しが減る |
| 隙間(seam)を見つける(Michael Feathers) | 批大郤·导大窾 | 隙間が *どこに* 在るかの哲学 |
| 「速く動き、ものを壊せ」 | 結び目では 怵然为戒 | 空洞では速く、骨では遅く |
| ボーイスカウト・ルール | 刀刃若新发于硎 | ファイルだけでなく、ツールチェーンと信頼の全体を鋭く保つ |

> **以无厚入有间，恢恢乎其于游刃必有余地矣。**
> 厚みなきものを隙間あるものへ差し入れれば、刃はゆとりをもって自在に遊ぶ。
> — 荘子『養生主篇』
